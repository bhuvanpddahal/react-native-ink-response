# React Native InkResponse

A performant, highly customizable Material Design ripple effect for React Native.

## Installation

```sh
npm install react-native-ink-response
```

## Usage

```js
import { InkResponse } from 'react-native-ink-response';

// ...
return (
  <InkResponse>
    <Text>Tap me</Text>
  </InkResponse>
);
```
