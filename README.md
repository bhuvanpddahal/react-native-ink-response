# React Native InkWell

A performant, highly customizable Material Design ripple effect for React Native.

## Installation

```sh
npm install react-native-ink-well
```

## Usage

```js
import { InkWell } from 'react-native-ink-well';

// ...
return (
  <InkWell>
    <Text>Tap me</Text>
  </InkWell>
);
```
